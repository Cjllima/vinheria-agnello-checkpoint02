Implementações no Checkpoint 2:

Foi implementado funções de verificação para auxiliar no monitoramento de dados do tipo texto, ano e quantidade, adicionado uma variável contador para contar quantos cadastros foi realizado, uma variável para vinhos com estoque baixo para contar quantos estão com estoque baixo, junto com isso uma uma função para verificar e armazenar quantos vinhos tem estoque baixo, uma função para classificar o vinho, duas variáveis para armazenar qual o vinho mais antigo cadastrado e um console log com as informações de cadastros feitos, vinhos com estoque baixo e vinho com a safra mais antiga
